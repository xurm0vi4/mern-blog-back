# mern blog backend
Backend for mern-blog project. [Front-end part here.](https://github.com/xurm0vi4/mern-blog-front.git)

## Run guide

### Installing dependencies
To install the dependencies, run:
```sh
$ npm install
```

### Starting the server
To start the server, run:
```sh
npm run dev
```

## Technology stack
- NodeJs
- Express
- Express-validator
- Mongoose
- JWT
- Multer
- Bcrypt
- Cors
- Nodemon
